# deckparty JS

A JQuery plugin to manage decks for HTML5 apps


### Standalone
You can use the standalone version:
```html
<script src="deckparty.js"></script>
```

## How-to
Create a new instance of deckparty.
```js
$().deckparty({	deck : 'mydeck' });
```
